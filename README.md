# photoTutorial-ios
사진앨범을 활용해서 profile image 를 바꾸는 tutorial

- photo 관련 오픈라이브러리 : [YPImagePicker]https://github.com/Yummypets/YPImagePicker
- YPImagePicker 오픈라이브러리를 활용해 사진앨범에서 사진을 가져와 프로필이미지로 바꾸어보았다.
